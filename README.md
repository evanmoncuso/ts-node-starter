# TS-Nose Starter
A premade configuration for a TS node app. This way I can write it once and rest on my laurels for the rest of time.

## Dependencies
 - Express (router, etc)
   - BodyParser
   - CORS
 - Jest (testing)
 - TS Lint (linting)
 - Nodemon, Concurrently (lazy dev builds)
